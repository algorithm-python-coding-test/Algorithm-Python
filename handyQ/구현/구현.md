## 시각

- 00시 일 때 3이 포함되는 경우의 수
- 00시 전체 경우의 수
- 3,13,23 일떄는 전체 경우의 수 더하고 나머지는 3이 포함되는 경우의 수 더하기

```python
h = int(input())

# 00시 일때 3이 포함되는 경우의 수
cnt=0
for minute in range(60):
    for second in range(60):
        if '3' in str(minute)+str(second):
            cnt+=1
# print(cnt) = 1575

# 1시간 일때 경우의 수
# print(60*60) = 3600


#
ans=0
for hour in range(h+1):
    if hour ==3 or hour ==13 or hour==23:
        ans += 3600
    else:
        ans += 1575

print(ans)
```

## 게임 개발

- 방향 은 왼쪽으로   북 -> 서 -> 남 -> 동

- 가본칸 visited 체크,

- 바다 면 break ->  전부 해본 것. (1 이거나 갈 곳 없는 경우)

  

```python
# 회전
def turn():
    global dirc
    dirc += 1
    if dirc == 3:
        dirc =0

N, M = map(int,input().split())  # 세로, 가로

visited = [[0 for _ in range(M)] for _ in range(N)]

# print(visited)

i, j , dirc = list(map(int,input().split()))   # 현재좌표, 방향  (북0 서3 남2 동1)
dirc[i][j] = 1


lst = []
for n in range(N):
    lst.append(list(map(int,input().split())))
# print(lst)


# 복 서 남 동
di = [-1,1,0,0]
dj = [0,0,1,-1]




cnt = 1
turning = 0

while True:
    turn()
    ni = i + di[dirc]
    ny = j + dj[dirc]
    
    if visited[ni][nj] ==0 and lst[ni][nj] == 0:
        visited[ni][nj] =1
        i = ni
        j = nj
        cnt +=1
        continue
    else:
        turning +=1
        
    
    if turning ==4:
        ni = i - di[dirc]
        nj = j - dj[dirc]
        
        if lst[ni][nj] ==0:
            i = ni
            j = nj
        
        else:
            break
        turning=0

print(cnt)
```
